# Sistema de Ticketing de Incidencias LAB-LIS

#  Descripción del proyecto

Este proyecto es el desarrollo de un **Sistema de Ticketing (gestión de incidencias)** por consola, implementado en **Java** bajo el paradigma de la **Programación Orientada a Objetos (POO)**.

El objetivo principal es gestionar las fallas técnicas (hardware, software, red) de los equipos de cómputo del Laboratorio LAB-LIS.

El sistema permite:
* Registrar nuevas incidencias y clasificarlas por tipo (hardware, software, red).
* Asignar prioridades y gestionar el ciclo de vida de la incidencia (asignar técnicos, cambiar estado).
* Consultar, filtrar incidencias y generar reportes básicos del estado del laboratorio.

# Explicación para la Ejecución del Programa

Para ejecutar este programa, asegúrate de tener instalado el **Java Development Kit (JDK)**.

# Pasos para Ejecución:

1.  **Clonar el Repositorio:**
    
    git clone https://github.com/JOSE19291/Proyecto_Progrmacion_orientada_a_objetos.git
    

2.  **Abrir en tu IDE:**
    * Abre el proyecto en tu entorno de desarrollo en este caso VSC.

3.  **Ejecutar:**
    * Localiza el archivo **"APP.java"**.
    * Ejecuta el método "app".

El sistema se ejecutará inmediatamente en la consola, mostrando una **demostración secuencial** de todas sus funcionalidades (registro, asignación, filtrado y reportes).
